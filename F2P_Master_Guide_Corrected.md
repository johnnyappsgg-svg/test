This is an **offerwall rewards guide.** The dollar amounts listed throughout are **what you earn** by completing milestones — not what you spend. "F2P achievable" means you can complete the task without spending real money on the game. "Requires real money" (💸) means the task payout is not worth the spend required to complete it efficiently.

## 🧭 THE BIG IDEA

This guide is built around one rule: early days should produce fast returns, while later days should only be used for high-value long timers.

The games do not all scale the same way.

Some games pay out early with very little effort. Some games require strict account focus. Some are timer games where you win by never leaving builders idle. Some are grind games where one carry character or one city path matters more than everything else.

The best route is:

1. Hit the fastest milestones first
2. Keep long timers always running
3. Dump resources only when they create milestone progress
4. Stop before bad-value walls
5. Never spread resources thin

## 🎓 FIRST-HOUR TUTORIALS / SETUP

---

### 1️⃣ Idle Weapon Shop — Tutorial Priority

This game is about storage, flow, and timing.

**First actions**
1. Finish the opening tutorial.
2. Upgrade Storage first.
3. Hire your first crafter.
4. Hire your first seller.
5. Keep production moving.
6. Push story missions until they stop giving clean progress.

**Why this matters**

If storage is too low, production stalls. If production stalls, income collapses. If income collapses, all other upgrades slow down.

**Early pattern**
- Do not overbuild weak weapons.
- Do not fill inventory with slow-selling junk.
- Keep high-demand items circulating.
- Use ads only when they are directly increasing income.

**Early mistake to avoid**

Do not chase speed upgrades before storage is healthy. That creates a bottleneck that makes the game feel slower instead of faster.

---

### 2️⃣ Game of Vampires: Twilight Sun — Tutorial Priority

This is an event-timing game. Early efficiency matters more than raw spending.

**First actions**
1. Join the **freshest available server** (prioritize a server that opened in the last 1–3 days).
2. Spend all Blood on your starter Warden.
3. Use single upgrades only for Twilight Bonus.
4. Push Exploration immediately.
5. Join a Guild as early as possible.

**Why this matters**

This game rewards one strong growth path, not a scattered account. Your best route is to concentrate resources into one carry and use events to multiply gains.

**Early pattern**
- Build only what pushes Dominance.
- Keep early non-core stats low.
- Do not waste upgrade material on side characters.
- Save key items for event windows.

**Best starter path: Ulfred**

Ulfred is the best starter Warden for this route because he gives the cleanest Strength-focused return and fits the event dump strategy.

---

### 3️⃣ Call of Dragons — Tutorial Priority

This is a builder efficiency and command point game.

**First actions**
1. Join an Alliance immediately.
2. Use the second builder trial right away.
3. Rush the Architecture research nodes (levels 1–5), then Scholarship level 1, then continue the Architecture research chain.
4. Keep builders active nonstop.
5. Start gathering as soon as the map opens.

**Why this matters**

This game is about never wasting build time, research time, or Command Points. The account gains power by constant motion.

**Early pattern**
- Keep Hall prerequisites moving.
- Use the free buffs while they are active.
- Kill Darklings whenever Command Points are available.
- Research the shortest useful path first.

**Early mistake to avoid**

Do not leave builders idle while "thinking about" the next step. In this game, idle time is lost progress.

---

### 4️⃣ Raid: Shadow Legends — Tutorial Priority

This is a single-carry account game.

**First actions**
1. Finish the tutorial.
2. Claim every free reward.
3. Pick Kael and invest only in Kael.
4. Push campaign progression.
5. Save food champions for rank-up.
6. Use multi-battle early for efficiency.

**Why this matters**

Raid punishes resource dilution. If you spread upgrades too early, you delay the one champion that actually carries the account.

**Early pattern**
- Kael is the account.
- Campaign progression is more important than random leveling.
- Food champions should be preserved, not wasted.
- Gem usage should be targeted only when it produces direct power.

---

### 5️⃣ Puzzles & Chaos: Frozen Castle — Tutorial Priority

This is a castle path and prerequisite game.

**First actions**
1. Finish the tutorial and core missions.
2. Build only prerequisites.
3. Upgrade only what the next Castle level requires.
4. Join the strongest active Alliance available.
5. Use puzzle stages for materials and speedups.

**Why this matters**

The game rewards progress discipline. If you build extra things too early, you slow down Castle growth and weaken your route.

**Early pattern**
- Every action should be helping the next Castle target.
- Puzzles are not the main objective by themselves.
- Speedups are valuable only when they move the Castle forward.

---

### 6️⃣ Rise of Kingdoms — Tutorial Priority

**Civilization: China** ✅ (already chosen — correct choice)

China's bonuses: +5% gathering speed for all resource types, faster starting units, and Sun Tzu as a free starting commander who is one of the best F2P options in the game.

**First actions**
1. Rush City Hall 3, 5, and 8.
2. Complete all tutorial quests.
3. Build required resource buildings immediately.
4. Keep both builders active at all times.
5. Join an Alliance within the first 30 minutes.

**Why this matters**

RoK is about speedup conversion. Everything should be converted into City Hall progression or important research.

**Early pattern**
- Don't waste gems early.
- Don't spend speedups on weak buildings.
- Keep all troops gathering whenever possible.
- Use alliance help constantly.
